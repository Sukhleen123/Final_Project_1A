How to Play


In this game, the user is posed with a scenario and two different ways they can approach that situation. Either option has a survival percentage, or how likely the user will survive if they choose that answer. When the user chooses an option, the program will run a random number generator(from a range of 1-10 or 1-15), and that number will be used to determine whether the user moves on to the next question or not. For example, let's say one option has a survival percentage of 70% and the random number generator has a range of 1-10. That means if a number from 1-7 is generated, the user will continue to the next screen. If a number from 8-10 is generated, the user will be directed to the game over screen. Some options have survival percentages of 100% and 0%, so it is up to the user to realize when that is the case. Furthermore, the survival percentage of each option are not affected by decisions the user makes throughout the game. Enjoy!
